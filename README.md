# 動画重複削除システム

指定したフォルダ内の重複動画ファイルを検出し、削除（ゴミ箱へ移動）するためのWebアプリケーションです。

## 機能
- 指定フォルダ（サブフォルダ含む）の動画ファイルをスキャン
- ファイルサイズとハッシュ値（部分ハッシュ）による重複検出
- 重複ファイルのリスト表示
- 選択したファイルの削除（ゴミ箱への移動）

## セットアップと実行

1. 依存関係のインストール:
   ```bash
   npm install
   ```

2. サーバーの起動:
   ```bash
   node server.js
   ```
   または
   ```bash
   npm start
   ```
   (package.jsonにstartスクリプトを追加する必要があります)

3. ブラウザでアクセス:
   [http://localhost:3000](http://localhost:3000)

## 注意事項
- 動画ファイルはサイズが大きい傾向があるため、パフォーマンスを考慮してファイルの先頭・中間・末尾のデータのみを使用してハッシュ値を計算しています。
- 削除されたファイルはシステムのゴミ箱に移動されますが、念のため重要なファイルはバックアップを取ってから操作してください。
